### Trénink se zbraní

**Omezení:** -

**Opakování:** Lze vzít opakovaně pro různé zbraně

Stáváš se trénovaný se zbraní svého výběru. Za předpokladu, že základní kostka poškození této zbraně není vyšší než nejvyšší základní kostka poškození zbraně, kterou již ovládáš.

<img src="/assets/sep_line.png"/>

### Trénink se zbrojí nebo štítem

**Omezení:** -

**Opakování:** Lze vzít opakovaně pro různou výzbroj

Vyber si zda se chceš stát trénovaný se zbrojí nebo se štítem.

- **Zbroj:** Stáváš se trénovaný s o jeden stupeň vyšší kategorií zbrojí než si aktuálně trénovaný. Například pokud nejsi trénovaný s žádnou zbrojí, stáváš se trénovaný s lehkou zbrojí a podobně.
- **Štít:** Stáváš se trénovaný s o jeden stupeň vyšší kategorií štítu než si aktuálně trénovaný. Například pokud nejsi trénovaný se štíty vůbec, stáváš se trénovaný s pukléři.
